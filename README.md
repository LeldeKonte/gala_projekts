## Projekts

Šajā projektā izmanto TMDB datubāzi un LLM datu analīzei.

Sistēma:
- pieslēdzas SQL serverim,
- iegūst datus,
- izveido grafikus,
- izveido reportu,
- izmanto Docker Compose,
- izmanto Streamlit UI.



## Palaišana

Instalē bibliotēkas:

pip install -r requirements.txt


Palaid projektu:

streamlit run app.py


Docker:

docker compose up
